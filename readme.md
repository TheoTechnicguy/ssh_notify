# SSH Notify

This project is an attempt at a ssh-connection notification system.

# Installation

For the moment, this project is in _dev-only_ mode. If you wish to contribute, you can

```bash
git clone https://github.com/TheoTechnicguy/ssh_notify.git
```

## Requirements

I use `pytest` for testing. You can get all requirements for testing via

```bash
pip install -Ur test-requirements.txt
```
